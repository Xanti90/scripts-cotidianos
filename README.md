# Scripts Cotidianos 🛠️

Colección de scripts de automatización para tareas del día a día.

## Estructura

| Carpeta | Contenido |
|---------|-----------|
| `archivos/` | Scripts para organizar, renombrar, mover o limpiar archivos |
| `sistema/` | Scripts de mantenimiento y monitoreo del sistema |
| `web/` | Scripts para descargas, scraping y tareas web |
| `utilidades/` | Scripts de uso general y conversiones |

## Cómo usar

Cada script incluye instrucciones en su interior.
Ejecutar desde terminal: `python3 nombre_script.py`
